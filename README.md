# SlidingView
右滑解锁，继承自RelativeLayout

# Usage
```java
     <com.linxk.lib.MySlidingView
        android:id="@+id/my_sliding_view"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:orientation="vertical">
        ...
    </com.linxk.lib.MySlidingView>
```
